### To use this overlay please do the following on Funtoo:
### as sudo or su

--------------------------------------------------------------------

* cd /var/git
* git clone https://github.com/bradlyatc/funtoo-overlay
* cd funtoo-overlay
* cp funtoo-overlay /etc/portage/repos.conf
